О программе

Данная программа предназначена для автоматизации учёта успеваемости студентов с поддержкой поурочного учёта. Программа позволяет:

  • Создавать и управлять группами студентов
  • Добавлять предметы для каждой группы
  • Вести список студентов с автоматическим присвоением ID
  • Проводить уроки с фиксацией даты, темы и домашнего задания
  • Выставлять несколько оценок одному студенту за урок
  • Просматривать журнал успеваемости с группировкой по датам
  • Смотреть историю оценок каждого студента
  • Анализировать статистику и формировать рейтинг
